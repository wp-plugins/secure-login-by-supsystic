== Description ==

Simple and easy way to to secure login page with Loging Security by Supsystic:

* Brute force attack protection with permanent and temporary black lists
* Track users login IP, Country, date and time
* Block login by IP, Country, language
* Integrated login protection against unauthorized access

== Installation ==

= To install a login security plugin via FTP =

1. Download the plugin
2. Unarchive the plugin
3. Copy the folder with plugin
4. Open ftp \wp-content\plugins\
5. Paste the plug-ins folder in the folder
6. Go to admin panel => open item "Plugins" => activate Login Security by Supsystic

NOTE: Dont forget to using backup as the one of the most effective way to defence your website
